# 科学上网翻墙梯子机场VPN

## 前言

三个VPN都是使用过相对便宜性价比高的。
魔戒和飞鸟云更推荐按流量付费，Cocoduck是传统的按月付费。
速度稳定性方面，当然越贵越稳定. 
Cocoduck > 魔戒 > 飞鸟云
